# Estrutura de Diretórios

```
api/         # Endpoints REST e integrações externas
config/      # Arquivos de configuração (DB, segurança, servidor)
cron/        # Scripts de tarefas agendadas
css/         # Estilos CSS globais
ecm/         # Módulos de gestão documental
includes/    # Núcleo da aplicação (classes, funções, libs, internacionalização)
js/          # Scripts JavaScript
modules/     # Módulos adicionais
plugins/     # Plugins e extensões
template/    # Templates e temas do frontend
uploads/     # Uploads de usuários
log/         # Logs do sistema
cache/, tmp/ # Armazenamento temporário
```
