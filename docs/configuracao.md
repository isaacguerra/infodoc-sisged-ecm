# Configuração Inicial

1. **Pré-requisitos:** PHP 7.4+, MySQL/MariaDB, Apache/Nginx.
2. **Configuração do Banco:** Editar `config/database.php` com as credenciais.
3. **Permissões:** Pastas `uploads/`, `cache/`, `tmp/`, `log/` devem ser graváveis pelo servidor web.
4. **Instalação:** Importe o banco de dados conforme instruções do arquivo SQL fornecido.
5. **Acesso:** Navegue até `http://localhost/infodoc-sisged` ou o domínio configurado.
