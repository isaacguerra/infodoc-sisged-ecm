# Customização e Desenvolvimento

## Novos Módulos
Crie uma pasta em `modules/` e siga o padrão dos existentes para adicionar novas funcionalidades.

## Plugins
Adicione em `plugins/` e registre no sistema conforme documentação interna.

## Templates
Edite arquivos em `template/` para personalizar o visual da aplicação.

## Traduções
Adicione ou edite arquivos em `includes/languages/` para novos idiomas ou ajustes.
